# Pure Driving - 驾考科目一纯净刷题 Web App

![Logo](./public/logo.svg)

一个基于 React + TypeScript 构建的轻量级驾考科目一练习应用。
主打无广告、高性能与流畅的交互体验。支持移动端与桌面端。

## 核心功能

### 刷题体验
- **交互优化**：底部操作区固定布局，解决长题目导致的按钮跳动问题。
- **自动流模式**：支持“答对自动下一题”，减少重复点击，提升练习效率（可配置开关）。
- **键盘支持**：支持方向键翻页、数字键选题，双手不离键盘完成全部操作。
- **多题库支持**：内置“精选 500 题”与“完整题库”，支持一键切换。

### 练习模式
- **顺序练习**：按题号顺序练习，自动记录当前进度。
- **随机练习**：优先抽取未答题目，支持错题回顾与历史回溯。
- **模拟考试**：还原真实考试规则（45分钟/100题/90分及格），支持倒计时、答题卡与自动判分。
- **错题本**：错题自动入库。提供“移除”机制，仅当用户确信掌握后移出。
- **题目屏蔽**：支持隐藏质量差或解析错误的题目。

### 工具特性
- **本地持久化**：利用 IndexedDB (LocalForage) 自动保存所有进度、设置与错题记录。
- **一键截图**：利用 html2canvas 支持题目区域一键生成图片，便于分享求助。
- **响应式 UI**：基于 Tailwind CSS，完美适配各类屏幕尺寸。

## 技术栈

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, Lucide React
- **State/Storage**: Context API, LocalForage
- **Scripts**: Python (用于题库数据的清洗与结构化处理)

## 快速开始

1. **环境准备**
   需要 Node.js 16+ 环境。

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发环境**
   ```bash
   npm run dev
   ```

4. **构建生产版本**
   ```bash
   npm run build
   ```

## 项目背景

最初是为女朋友备考科目一开发的个人工具。
旨在解决市面上相关 App 广告繁多、交互卡顿的问题。希望能为同样需要备考的朋友提供一个纯净、高效的备选方案。
